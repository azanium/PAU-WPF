1. Install xampplite-win32-1.7.3.exe atau xampp-win3-1.7.3.exe yang ada di dalam folder xampp
2. Buka XAMPP For Windows di menu Start -> All Programs -> XAMPP for Windows -> XAMPP Control Panel
3. Centang SVC di samping MySQL kemudian klik start di samping MySQL.
4. Install Sql Yog, yang ada di folder SQL Yog
5. Buka Sql Yog pada menu start -> All Programs -> Sql Yog -> Sql Yog
6. Setelah tampil window sql yog, pilih new connection, kemudian klik connect. Kemudian ikuti prosedur impor data.
7. Buka SQL Yog, dan import database sql yang ada di folder DB.
8. Kemudian install setup.exe.
